# Claude Code 分析文档展示页设计文档

**日期:** 2026-04-01
**状态:** 已批准

## 概述

创建一个赛博朋克风格的单页面HTML展示页，用于介绍Claude Code项目分析文档。页面采用分屏布局，支持多层次受众的信息展示。

## 目标受众

- **开发者** - 需要技术细节和代码示例
- **技术/产品经理** - 需要架构概览和功能说明
- **通用用户** - 需要简洁易懂的功能介绍

## 视觉风格：赛博朋克

### 配色方案

| 用途 | 颜色 | Hex值 |
|------|------|-------|
| 背景渐变起点 | 深黑 | `#0a0a0f` |
| 背景渐变终点 | 深紫 | `#1a0a2e` |
| 主色调1 | 霓虹粉 | `#ff00ff` |
| 主色调2 | 霓虹蓝 | `#00ffff` |
| 强调色1 | 酸性绿 | `#39ff14` |
| 强调色2 | 电光黄 | `#ffff00` |
| 文字主色 | 白色 | `#ffffff` |
| 文字辅色 | 浅灰 | `#a0a0a0` |

### 视觉效果

1. **发光边框** - 使用 `box-shadow` 创建霓虹发光效果
2. **文字阴影** - `text-shadow` 实现文字发光
3. **扫描线动画** - 模拟CRT显示器的扫描线效果
4. **故障艺术** - CSS动画实现的标题Glitch效果
5. **悬停脉冲** - 鼠标悬停时的高亮脉冲动画
6. **动态背景** - 可选的粒子/网格背景动画

## 页面布局

### 结构

```
┌─────────────────────────────────────────────────────────┐
│                    HEADER（全宽）                         │
│  Logo + Glitch标题 + 导航                                │
├──────────────┬──────────────────────────────────────────┤
│              │                                           │
│   SIDEBAR    │            MAIN CONTENT                  │
│   (固定)     │          (可滚动区域)                     │
│              │                                           │
│  • 导航菜单  │  ┌─────────────────────────────────┐    │
│  • 快速链接  │  │      项目概览                    │    │
│  • 目录结构  │  ├─────────────────────────────────┤    │
│              │  │      技术栈展示                  │    │
│  [发光边框]  │  ├─────────────────────────────────┤    │
│              │  │      架构分析                    │    │
│              │  ├─────────────────────────────────┤    │
│              │  │      文档导航网格                │    │
│              │  └─────────────────────────────────┘    │
└──────────────┴──────────────────────────────────────────┘
```

### 响应式断点

- **移动端** (< 768px): 单栏布局，侧边栏收起为汉堡菜单
- **平板** (768px - 1024px): 侧边栏可折叠
- **桌面** (> 1024px): 固定侧边栏 + 主内容区

### 核心CSS效果

**发光效果:**
```css
.neon-border {
  box-shadow: 0 0 5px #ff00ff, 0 0 10px #ff00ff, 0 0 20px #ff00ff;
}

.neon-text {
  text-shadow: 0 0 5px #00ffff, 0 0 10px #00ffff, 0 0 20px #00ffff;
}
```

**扫描线效果:**
```css
.scanlines::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: repeating-linear-gradient(
    0deg,
    rgba(0, 0, 0, 0.15),
    rgba(0, 0, 0, 0.15) 1px,
    transparent 1px,
    transparent 2px
  );
  pointer-events: none;
}
```

**Glitch动画:**
```css
@keyframes glitch {
  0% { transform: translate(0); }
  20% { transform: translate(-2px, 2px); }
  40% { transform: translate(-2px, -2px); }
  60% { transform: translate(2px, 2px); }
  80% { transform: translate(2px, -2px); }
  100% { transform: translate(0); }
}
```
