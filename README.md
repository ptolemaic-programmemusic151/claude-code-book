## 🚀 What this app does

claude-code-book helps you turn Claude Code output into structured books and AI-ready context files for development work. It is built for people who want a simple way to save, organize, and reuse Claude Code content on Windows.

Use it when you want to:

- Collect Claude Code notes in one place
- Turn long output into a cleaner book format
- Keep useful context for later work
- Make project notes easier to read
- Reuse content without sorting through long chat logs
